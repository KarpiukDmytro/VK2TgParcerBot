bot:
    VKWallPostParser - основной класс для парсинга данных со стены и подготовки их к отправке в телеграм

Configurations:
    BotConfig - класс для создания бина телеграм-бота
    PostData - класс, собирающий данные записи со стены в один объект для удобства дальнейшей работы

services:
    LastPostIDService - сервис, запоминающий ID последнего поста со стены в VK и перезаписывающий его
    PostSchedulerService - класс для регулярной проверки новых постов в VK и их отправку в Telegram
    TelegramPostPublisher - обработка полученных постов из VK и подготовка их отправки в телеграм
    VKWallPostParser - класс для парсинга новых записей и сбора данных из них со стены сообщества

WalleBotApplication - основной класс для запуска приложения
